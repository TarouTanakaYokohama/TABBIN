TABBIN is a WXT-based browser extension for saving, organizing, and managing browser tabs/URLs. Main entrypoints are in entrypoints/ including background.ts, options, saved-tabs, and changelog. Domain features are under features/, reusable UI under components/ and components/ui/, shared logic under lib/, shared types under types/, constants under constants/, utilities under utils/. Tech stack: TypeScript, React 19, WXT, Tailwind CSS v4, shadcn-style UI components, Vitest, Playwright, Biome, Bun.
