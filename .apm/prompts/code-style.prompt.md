# Code Style Review

TABBIN の既存コードと `AGENTS.md` のリポジトリガイドラインを基準に、
現在の変更差分を確認してください。

## 確認観点

- TypeScript と React の既存パターンに沿っているか。
- Biome の設定に合う import、命名、整形になっているか。
- 不要な抽象化、重複、広すぎる変更が入っていないか。
- テストが変更範囲に対して十分か。

## 出力

問題がある場合は、ファイルと該当箇所を示して修正案を簡潔に述べてください。
問題がなければ、その旨と残るリスクだけを短くまとめてください。
